# WebScrapper
A Python web scraping project that extracts book names and prices from a public website using Requests, BeautifulSoup, and Pandas, then stores the scraped data in a CSV file for analysis.
# Web Scraping Project using BeautifulSoup

## Overview
This project extracts book names and prices from a public website using Python.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Features
- Extracts book names
- Extracts book prices
- Creates a custom dataset
- Saves data as a CSV file

## Output
The scraped data is stored in `books.csv`.

## Website Used
https://books.toscrape.com/
